Software Engineering Course Syllabus

Course Description

Methods, strategies, and tools for implementing software systems, particularly as part of a development team. Topics include the software development life cycle, Unified Modeling Language, software testing techniques, software security, open-source development, requirements gathering and documentation, maintenance, and basic software project management. Prerequisites: CPSC-24500 Object-Oriented Programming

Academic Integrity

Students engaging in such activities are subject to loss of credit and expulsion from the University.
